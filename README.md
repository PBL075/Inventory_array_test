 Sistem de Inventar în C# (Array Test)

Acesta este un proiect simplu în C# care demonstrează utilizarea *Array-urilor* pentru a gestiona un sistem de inventar de obiecte. Programul permite stocarea, modificarea și afișarea elementelor din inventar folosind o buclă repetitivă.

 Funcționalități
* Stocare: Utilizează un tablou de tip `string[]` pentru a păstra numele obiectelor.
* Modificare: Demonstrează cum se poate actualiza un element specific folosind indexul (ex: transformarea unei puști simple într-una de nivel superior).
